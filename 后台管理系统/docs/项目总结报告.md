### 核心流程

#### 索引构建流程
```
1. 用户上传文档
   ↓
2. 保存文档到数据库（status: uploading）
   ↓
3. 提取文档内容（Python脚本）
   ↓
4. 智能分块（根据内容类型）
   ↓
5. 向量化（批量处理，带缓存）
   ↓
6. 合并到索引文件
   ↓
7. 更新数据库（status: indexed）
   ↓
8. 通知前端完成
```

#### 问答流程
```
1. 用户输入问题
   ↓
2. 标准化问题（口语→正式）
   ↓
3. 向量化问题
   ↓
4. 检索相关文档（余弦相似度）
   ↓
5. 重排序（可选）
   ↓
6. 去重（按页码）
   ↓
7. 生成答案（AI）
   ↓
8. 返回结果（带来源）
```

## 📊 智能分块策略

### 文档类型识别
```python
def detect_content_type(self, document):
    """检测文档内容类型"""
    if self.contains_keywords(document, ['政策', '规定', '办法']):
        return 'policy'
    elif self.contains_keywords(document, ['手册', '指南']):
        return 'handbook'
    else:
        return 'general'
```

### 分块策略
```python
strategies = {
    'policy': {
        'chunk_size': 800,
        'overlap': 150,
        'split_by': 'article'
    },
    'handbook': {
        'chunk_size': 1000,
        'overlap': 200,
        'split_by': 'chapter'
    },
    'general': {
        'chunk_size': 1200,
        'overlap': 200,
        'split_by': 'paragraph'
    }
```
