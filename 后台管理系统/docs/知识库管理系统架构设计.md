# 知识库管理系统架构设计

## 系统概述

### 设计目标
1. **灵活性**：支持多种大模型（智谱AI、OpenAI、阿里通义、百度文心等）
2. **可扩展性**：支持增量添加、替换、删除索引
3. **易用性**：一键操作，带进度条显示
4. **职责分离**：后端构建索引，前端使用服务

### 系统架构

### 3. 文档处理模块

#### 智能分块策略
```python
class IntelligentChunker:
    def analyze_document(self, document):
        """分析文档特征"""
        stats = {
            'total_pages': len(document['pages']),
            'avg_page_length': np.mean([len(p['text']) for p in document['pages']]),
            'has_chapters': self.detect_chapters(document),
            'content_type': self.detect_content_type(document)
        }
        return stats
    
    def determine_strategy(self, stats):
        """根据文档特征确定分块策略"""
        if stats['content_type'] == 'policy':
            # 政策文档：较小的块，保持条款完整性
            return {
                'chunk_size': 800,
                'overlap': 150,
                'split_by': 'article'  # 按条款分割
            }
        elif stats['content_type'] == 'handbook':
            # 手册文档：中等块，按章节分割
            return {
                'chunk_size': 1000,
                'overlap': 200,
                'split_by': 'chapter'  # 按章节分割
            }
        else:
            # 通用文档：默认策略
            return {
                'chunk_size': 1200,
                'overlap': 200,
                'split_by': 'paragraph'  # 按段落分割
            }
```

## 数据流设计

### 索引构建流程
```
1. 用户上传文档
   ↓
2. 保存文档到数据库（status: uploading）
   ↓
3. 提取文档内容（Python脚本）
   ↓
4. 智能分块（根据内容类型）
   ↓
5. 向量化（批量处理，带缓存）
   ↓
6. 合并到索引文件
   ↓
7. 更新数据库（status: indexed）
   ↓
8. 通知前端完成
```

### 问答流程
```
1. 用户输入问题
   ↓
2. 标准化问题（口语→正式）
   ↓
3. 向量化问题
   ↓
4. 检索相关文档（余弦相似度）
   ↓
5. 重排序（可选）
   ↓
6. 去重（按页码）
   ↓
7. 生成答案（AI）
   ↓
8. 返回结果（带来源）
```

## 安全性设计

### API密钥加密
```javascript
const crypto = require('crypto');

function encryptApiKey(apiKey) {
    const algorithm = 'aes-256-cbc';
    const key = Buffer.from(process.env.ENCRYPTION_KEY, 'hex');
    const iv = crypto.randomBytes(16);
    
    const cipher = crypto.createCipheriv(algorithm, key, iv);
    let encrypted = cipher.update(apiKey, 'utf8', 'hex');
    encrypted += cipher.final('hex');
    
    return {
        encrypted,
        iv: iv.toString('hex')
    };
}

function decryptApiKey(encrypted, iv) {
    const algorithm = 'aes-256-cbc';
    const key = Buffer.from(process.env.ENCRYPTION_KEY, 'hex');
    
    const decipher = crypto.createDecipheriv(algorithm, key, Buffer.from(iv, 'hex'));
    let decrypted = decipher.update(encrypted, 'hex', 'utf8');
    decrypted += decipher.final('utf8');
    
    return decrypted;
}
```
