## 📋 功能详细说明

### 文档列表

**显示信息：**
- 文档名称（带图标）
- 描述
- 标签（徽章样式）
- 状态（图标+文本）
- 统计信息（页数、chunks数）
- 更新时间

**操作按钮：**
- 待处理/错误状态：显示"处理"按钮
- 已索引状态：显示"查看"和"替换"按钮
- 所有状态：显示"删除"按钮

### 统计卡片

**四个卡片：**
1. 总文档数 - FileText图标
2. 已索引 - CheckCircle2图标（绿色）
3. 待处理 - Clock图标（灰色）
4. 错误 - AlertCircle图标（红色）

### 上传对话框

**表单字段：**
1. 文件选择（支持拖拽）
2. 文档名称（必填）
3. 文档描述（可选）
4. 标签（逗号分隔）

**验证：**
- 文件大小限制（50MB）
- 文件类型限制（.docx, .doc, .pdf）
- 必填字段检查

### 状态轮询

**机制：**
- 处理开始后自动启动轮询
- 每3秒查询一次状态
- 处理完成或失败后自动停止
- 5分钟后强制停止（防止无限轮询）

## 🔌 API集成

### 使用的API端点

```typescript
// 文档管理
GET    /api/documents                    // 获取所有文档
GET    /api/documents/:id                // 获取单个文档
POST   /api/documents/upload             // 上传文档
POST   /api/documents/:id/replace        // 替换文档
DELETE /api/documents/:id                // 删除文档
POST   /api/documents/:id/process        // 处理文档
GET    /api/documents/:id/status         // 获取文档状态

// 索引管理
GET    /api/index/status                 // 获取索引状态
```

## 🚀 使用指南

### 启动后台管理系统

```bash
# 进入后台管理系统目录
cd "E:\外包\教育系统智能体\后台管理系统"

# 启动前端（端口：5173）
cd frontend
npm run dev

# 启动后端（端口：3001）
cd backend
npm run dev
```

### 访问政策文档管理页面

1. 打开浏览器访问：`http://localhost:5173`
2. 点击左侧导航栏的"政策文档管理"
3. 开始管理你的政策文档

### 基本操作流程

**上传新文档：**
1. 点击"上传文档"按钮
2. 填写文档信息（名称、描述、标签）
3. 选择文件
4. 点击"上传"
5. 上传成功后点击"处理"按钮
6. 等待处理完成

**替换文档：**
1. 找到要替换的文档
2. 点击"替换"按钮
3. 选择新文件
4. 点击"替换"
5. 重新处理文档

**删除文档：**
1. 找到要删除的文档
2. 点击"删除"按钮
3. 确认删除
