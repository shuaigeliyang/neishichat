## 🔍 故障排查

### 问题1：Backend启动失败

**检查**：
```bash
# 检查端口占用
netstat -ano | findstr :3000

# 检查环境变量
type .env
```

**解决**：
- 如果端口被占用，关闭占用进程
- 如果环境变量错误，检查.env文件格式

### 问题2：前端无法登录

**检查**：
- Backend是否正常运行
- 用户名密码是否正确
- 浏览器控制台是否有错误

**解决**：
- 确认Backend运行在http://localhost:3000
- 尝试清除浏览器缓存
- 检查网络连接

### 问题3：RAG返回错误内容

**检查**：
```bash
# 检查索引文件
ls -lh ../文档库/indexes/unified_index.json

# 检查Python Embedding服务
curl http://localhost:5001/health
```

**解决**：
- 如果索引文件不存在，运行索引构建脚本
- 如果Python服务未启动，运行start-embedding-service.bat
